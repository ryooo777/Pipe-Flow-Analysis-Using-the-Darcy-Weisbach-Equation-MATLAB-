# Pipe Flow Analysis Using the Darcy–Weisbach Equation (MATLAB)

This MATLAB program analyzes pressure losses in fluid flow through pipes using the **Darcy–Weisbach equation**.  
Friction factors are computed using both the **Colebrook equation** and the **Haaland approximation**.

The program evaluates:

- Reynolds number
- Pressure drop
- Friction factor
- Head loss
- Pump power requirements

---

# Development

## Option 1: Using MATLAB Online (Recommended)
1. Go to **MathWorks** and create an account if you do not already have one.
2. Sign in and open **MATLAB Online**.
3. Download all files from this repository.
4. Open and run `mainFluids.m`.

## Option 2: Using MATLAB Desktop
1. Download all files from this repository.
2. Open **MATLAB** on your computer.
3. Open the script `mainFluids.m`.
4. Run the script.
